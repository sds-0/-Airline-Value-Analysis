Airline-Value-Analysis
本项目聚焦航空公司客户价值挖掘，基于经典 LRFMC 模型（客户关系长度 L、最近消费间隔 R、消费频率 F、消费里程 M、平均折扣系数 C），实现航空客户数据的全流程分析与价值分层。
核心流程：
数据预处理：通过数据清洗处理缺失值、异常值，构建标准化流程保障数据质量；
客户聚类：利用 K-Means 算法将客户划分为 4 类（重要保持 / 发展 / 挽留客户、一般与低价值客户）；
特征可视化：借助雷达图、饼图剖析不同客户群体的 LRFMC 特征差异；
策略输出：针对各客户类别制定针对性营销策略。
项目价值：
为航空公司提供高价值客户识别、资源优化配置的数据支撑，验证了数据挖掘在客户关系管理（CRM）中的应用价值；
标准化的分析流程、客户分层方法，可为航空业数字化运营、精准营销提供实践参考，助力提升客户忠诚度与企业收益。
